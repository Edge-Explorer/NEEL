## ğŸ¤– AI Agent Architecture

NEEL doesn't use a single prompt. It uses a cascading chain of agents to ensure high-fidelity coaching:

1.  **Supervisor Agent**: Analyzes the current user profile and activity logs. Decides if there is enough data to generate "High-Level Intelligence" or if the user needs more "Onboarding Guidance."
2.  **Reasoning Agent**: The core strategist. It looks at historical summaries, conversational context (permanent memory), and current goals to generate a draft response.
3.  **Reflection Agent**: The Auditor. It reviews the Reasoning Agent's draft for tone, safety, and goal alignment. It can reject, soften, or approve the advice.
